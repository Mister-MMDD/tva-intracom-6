Interface Streamlit du moteur TVA

Une interface modulaire pour piloter le moteur fiscal et les exports.

Onglets principaux

  • 💶 Déclarations : synthèse “Ce que vous devez reverser” (CA3, OSS, IOSS, immatriculations locales).
  • 📋 Détail ventes : vue ligne par ligne, séparation “Vous / Amazon / Douane”.
  • 🛡️ VIES : validation des numéros TVA, overrides, reclassification B2B/B2C.
  • 🔬 Audit Amazon : écarts TVA Amazon, mouvements de stock FBA.
  • 📥 Téléchargements : exports Excel, XML OSS, rapports CA3, rapports locaux, FEC.
  • 📊 Visualisations : TVA par pays, carte Europe, évolution mensuelle.

Accéder à l’interface de calcul

Le moteur est accessible via une application web sécurisée. Importez vos fichiers et obtenez vos rapports en quelques secondes.
