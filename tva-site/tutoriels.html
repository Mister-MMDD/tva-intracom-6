Guides & Cas pratiques

Retrouvez ci-dessous les guides thématiques et les cas concrets modélisés par le moteur. Chaque cas peut être déplié pour voir le détail des règles appliquées.

GUIDE

Livraison B2B

Comprendre quand une livraison est exonérée de TVA.

  1. Identifier fournisseur et client (États membres différents).
  2. Vérifier le numéro de TVA du client (VIES).
  3. Confirmer le transport vers l’autre État membre.
  4. Conserver les preuves de sortie.
GUIDE

Acquisition B2B

Appliquer l’autoliquidation correctement.

  1. Identifier une acquisition de biens entre deux États membres.
  2. Déterminer le pays du client (lieu d’imposition).
  3. Calculer la TVA due et déductible.
  4. Reporter les montants dans la déclaration.
GUIDE

Ventes B2C et OSS

Gérer les ventes à des consommateurs dans plusieurs pays.

  1. Suivre le chiffre d’affaires B2C par pays.
  2. Vérifier le seuil de 10 000 €.
  3. Identifier le pays du consommateur.
  4. Déclarer via OSS si applicable.
SCÉNARIO

DOMESTIC – Vente locale

Stock et acheteur dans le même pays UE.

Règle : TVA locale, canal CA3 ou immatriculation locale.

Traitement moteur : classification DOMESTIC, ventilation par taux, intégration dans CA3 ou rapport local.

SCÉNARIO

OSS_B2C – Vente B2C

Stock UE, acheteur dans un autre pays UE, B2C.

Règle : TVA du pays de destination, déclaration via OSS.

Traitement moteur : agrégation OSS par pays, génération XML OSS officiel + Excel.

SCÉNARIO

B2B_REVERSE_CHARGE

B2B intra-UE avec numéro TVA VIES valide.

Règle : exonération côté vendeur, autoliquidation chez l’acheteur.

Traitement moteur : classification B2B_REVERSE_CHARGE, note fiscale avec références légales, contrôle VIES.

SCÉNARIO

DEEMED_SUPPLIER

Vendeur hors UE ou import ≤ 150 € B2C.

Règle : Amazon est réputé collecteur, vente exonérée pour le vendeur.

Traitement moteur : séparation "Vous / Amazon / Douane" dans les visualisations, exclusion des montants collectés par Amazon des déclarations du vendeur.

SCÉNARIO

IMPORT & IOSS_DIRECT

Import > 150 € (douane) ou ≤ 150 € avec IOSS.

Règle : TVA d’importation (douane) ou TVA collectée via IOSS.

Traitement moteur : classification IMPORT_STANDARD ou IOSS_DIRECT, ventilation par canal (douane, IOSS), intégration dans les rapports.

SCÉNARIO

MONACO – Flux spécial

Stock en France, acheteur situé à Monaco (MC).

Règle : en vertu de la convention fiscale franco-monégasque de 1963, Monaco est traité comme du territoire français au plan de la TVA — ce n'est ni une vente intra-UE, ni une exportation.

Traitement moteur : Monaco est exclu par défaut des recherches "UE" classiques ; le moteur applique un cas spécial dans engine.py qui reclassifie automatiquement ce flux comme vente domestique française (TVA française standard, pas d'OSS, pas de VIES).
