Documentation technique

Architecture du moteur fiscal, des parsers Amazon et de l’interface Streamlit.

Checklist de Conformité

  • Vérifier la validité des numéros TVA B2B via l'onglet VIES.
  • S'assurer que le Pays de Siège est correctement configuré.
  • Vérifier les alertes de Stock FBA pour les immatriculations locales.
  • Réconcilier le CA HT Net avec votre comptabilité via l'export FEC.
  • Télécharger et conserver le Rapport Principal Excel pour chaque période.

Sources de données & API

  • VIES (EU) : Validation des numéros de TVA en temps réel.
  • BCE (SDW) : Récupération quotidienne des taux de change officiels.
  • DGFIP / Schémas XSD : Validation des fichiers XML OSS.

Cœur du Moteur

  • engine.py : Moteur de classification fiscale.
  • vies_engine.py : Validation VIES avec cache PostgreSQL.
  • ca3_report.py : Génération rapports CA3 (Cerfa).
  • fec_export.py : Export comptable (Journal des ventes).
  • oss_xml.py : Générateur XML OSS conforme 2021/965.

Scénarios modélisés

Scénario Situation Règle appliquée
DOMESTIC Stock et acheteur dans le même pays UE. TVA locale du pays.
OSS_B2C B2C transfrontalier intra-UE. TVA du pays de destination (OSS).
B2B_REVERSE B2B intra-UE avec n° TVA valide. Exonération (Autoliquidation).
DEEMED_SUPPLIER Vendeur hors UE ou import ≤ 150€. Collecté par Amazon (Exonéré).
EXPORT Acheteur hors UE. Exonération (Exportation).
IMPORT_STD Import > 150€ (B2C). TVA importation (Douane).
IOSS_DIRECT Import ≤ 150€ avec n° IOSS propre. Collecté par le vendeur via IOSS.
IMPORT_DDP Import > 150€ (Vendeur importateur). TVA locale pays destination.

Formats Amazon supportés

  • Format 1-2 : Anciens formats TSV/Activity Period.
  • Format 3-4 : Formats 2024-2025 (Tax Collection Responsibility).
  • Format 5 : Rapport fiscal détaillé V5 (Multi-juridictions).
